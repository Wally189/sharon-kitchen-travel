How to update this site Simple steps for text, images and colours
  1. Change the visible wording below for headings, services, questions or contact details.
  2. Choose a site theme to switch the colour direction for both the public site and this editor.
  3. To change a photo, click into an image field first, then upload a file or press Use image in the library.
  4. Press Save website changes and rebuild, then open View site to check the live preview.

Security note. This admin is still using the fallback admin / playtrix login. Add a real .env file before any live use.
